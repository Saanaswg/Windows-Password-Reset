Usage:

1. Boot off Windows installation media with the reset files on a seperate USB
2. Enter the Command Prompt under the Repair option
3. Locate which drive letter your USB was assigned to, and run prep.bat 
4. Reboot your PC without the installation media, the USB with the reset files is nolonger required either
5. On the login screen, click on the "Ease of Access" button, it's location varies on different versions of Windows
6. Enter the username of the user that requires a password reset
7. Log in. If you are prompted for a password then leave it blank and hit enter

This code cleans itsself up.
